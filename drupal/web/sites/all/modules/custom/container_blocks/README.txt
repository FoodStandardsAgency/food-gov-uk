Name: Container Blocks Module
Author: Robert Castelo
Drupal: 7.x
********************************************************************
DESCRIPTION:

Provide blocks that contain other blocks so that we can for instance add a border around 3 blocks
and a gradient background.


********************************************************************
INSTALLATION:

Note: It is assumed that you have Drupal up and running.  Be sure to
check the Drupal web site if you need assistance.

1. Place the entire module directory into your Drupal directory:
   sites/all/modules/


2. Enable the module by navigating to:

   administer > modules

  Click the 'Save configuration' button at the bottom to commit your
  changes.
