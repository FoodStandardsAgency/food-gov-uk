Name: Facets Select Module
Author: Robert Castelo <www.codepositive.com>
Drupal: 7.x
********************************************************************
DESCRIPTION:

Provides custom features for FacetAPI facets:

* display date range facets as a select
* display field labels above selects, for better accessibility



********************************************************************
INSTALLATION:

Note: It is assumed that you have Drupal up and running.  Be sure to
check the Drupal web site if you need assistance.

1. Place the entire module directory into your Drupal directory:
   sites/all/modules/


2. Enable the module by navigating to:

   administer > modules

  Click the 'Save configuration' button at the bottom to commit your
  changes.



********************************************************************
CONFIGURATION:

Add Preserve input filter to input formats of your choice.
